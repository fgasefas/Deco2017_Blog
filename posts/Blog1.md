## Initial Concept

At the beginning, I first came up with a concept:

Right now in Australia, there is not a secondhand trading platform for Chinese international students, but the demand is actually high. People often trade used or unwanted items on Xiaohongshu (Rednote), but transactions there are basically built on trust, the information is messy, and it is easy for disputes to happen. So I want to design a more secure secondhand marketplace where, after payment is made, that portion of the money is first held by the platform, and the seller can only receive the money after the buyer confirms they have received the item. At the same time, there should also be customer support to handle transaction disputes.

## Direction and Core Features

Then we confirmed our design direction: a second-hand marketplace platform for international students, and we confirmed five core features:

1. Posting second-hand listings  
2. Categorizing and filtering items  
3. Viewing item detail pages  
4. Allowing interaction between users  
5. Personalized features  

## Page Structure

Then we confirmed the final main page categories:

1. Home page  
2. Category page (appears after searching)  
3. Post/Edit listing page  
4. Item detail page  
5. Chat/Messages page  
6. Profile page  

## Wireframes (My Part)

Then when we were making the wireframes, we split the work. I was responsible for the Messages pages and the Profile page.

### Messages Page — Conversation List

I started by drawing the Messages page. First I drew a message list view: the left side is the navigation bar, and the right side shows all recent conversations. Each conversation card includes the seller’s avatar, the seller’s display name, and a message preview. On the far right, it shows the message time and the number of unread messages; if everything is read, it only shows the time.

![Message page 1](/Deco2017_Blog/assets/images/Message-page-1.png)

### Messages Page — Chat View

The second Messages screen appears after you tap into a seller’s conversation. At the top there is the item information and the item image. Below that is the chat content, including read/unread status. Under the chat, I added quick replies and a message input field.

![Message page 2](/Deco2017_Blog/assets/images/Message-page-2.png)

### Profile Page

Finally, I drew the Profile page. The first thing you see is the user’s own avatar and display name. Below that are tabs for My Listings / Favourites / My Messages / Notifications. By default it shows My Listings, where you can see the user’s posted item image, item name, and price. On the right side, the user can edit their listings to easily mark an item as sold or remove it from the listing.
